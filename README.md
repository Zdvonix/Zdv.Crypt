Zdv Crypt - Это простой побайтовый XOR шифратор текста, прогоняемый через Base64.
---

## Принцип работы

### Шифрование

1. Ввод Текста
2. Ввод Пароля
3. Побайтовое XOR шифрование с использованием пароля
4. Кодировка Base64
5. Вывод зашифрованного сообщения

### Дешифрование

1. Ввод зашифрованного текста
2. Ввод Пароля
3. Декодировка Base64
4. Побайтовая XOR расшифровка с использованием пароля
5. Вывод зашифрованного сообщения

## Вероятные проблемы

После копирования текста нажмите куда-нибудь но не в консоль!

Иначе у вас не получится скопировать тккст

Повторяющиеся пароли будут работать одинаково

Пример: 123, 123123, 123123123 или 10, 1010, 101010
